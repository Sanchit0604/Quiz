# BrainBurst — Interactive Quiz App

A fully functional multiple-choice quiz application built with vanilla HTML, CSS, and JavaScript.

## Project Structure

```
interactive_quiz/
├── index.html    ← Quiz layout: intro, quiz, result screens
├── style.css     ← Full styling with CSS variables & animations
├── script.js     ← Quiz logic, data, rendering, scoring
└── README.md     ← This file
```

## Features

| Feature | Task |
|---------|------|
| Intro screen with quiz rules | Task 1 |
| Quiz layout with question + 4 options + Next button | Task 2 |
| Styled cards, animations, responsive layout | Task 3 |
| 10 questions stored in JS array, score tracking, navigation | Task 4 |
| Result screen with score, percentage bar, dynamic feedback | Task 5 |
| Restart Quiz button resets all state | Task 6 Bonus |

## How to Run

**Option A — Live Server (recommended)**
1. Open `interactive_quiz/` folder in VS Code
2. Install **Live Server** extension (Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**

**Option B — Direct**
- Double-click `index.html` in any modern browser
